Robot T-Shirt Design

a fun personal project to raise money for a youth robotics class

Project Goals

Create a robot-themed t-shirt design that:

  • is fun
  • is retro
  • is designed well using a consistent style

Initial Concepting

After doing a lot of research on toy tin robots, I sketched out a variety of visual elements including facial features and diagnostic guages.

Picking a Direction

I thought it would be fun to show a variety of different robot faces. I found many different facial features in my research and used them to create nine original faces.

Refining the Design

I chose to use only two line thicknesses across the whole project: a thicker line for outlines and large features and a thinner line for inner features and accents. This strategy unified all the robots into one cohesive design.

Final Product

The final design was unique, fun, and whimsical. It sold really well and got a lot of positive feedback!
