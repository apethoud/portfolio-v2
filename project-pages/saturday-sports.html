Saturday Sports Logo

a fun personal project to redesign the logo of a kids' sports club that I used to run

Project Goals

Create a new logo that:

  • represents how fun the program is
  • is similar visually in some way to the current logo
  • offers a beautiful, easy-to-use user interface

Initial Concepting

I sketched out different possible ways to use the "square in perspective" shape from the original logo in a new way.

Picking a Direction

I decided to create a custom-made font for the logo and position it diagonally.

Refining the Design

I outlined letters that overlapped others, to maintain readability.

Final Product

The final logo conveys a sense of movement and fun, while also loosely alluding to the shape of a baseball diamond.
