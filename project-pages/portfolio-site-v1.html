Personal Portfolio Site v1.0

a personal finance app designed to be forward-thinking and accurate

Project Goals

Create a personal finance app that:

  • helps people plan better for the future
  • provides an accurate gauge of their current financial situation
  • offers a beautiful, easy-to-use user interface

Initial Concepting

After brainstorming which features I wanted the app to have, I sketched out some user interface designs. A simple and clean visual style would best convey the professional and secure nature of the app.

Picking a Direction

Once I decided how the features should be organized within the app, I decided on a visual style for the app that utilized custom-made icons and a purple color scheme. These gave the impression of a professional, yet personal experience that would set us apart from our direct competitors.

Refining the Design

Consistently applying the visual style to each feature, popup, and button kept the app professional, easy-to-use, and pleasing to the eye.

Final Product

The final app provided easy-to-use features within a visually-pleasing and intuitive interface.
